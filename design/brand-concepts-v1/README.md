# Chai Studio Brand Concepts V1

These files are ImageGen concept samples for review. They are not production SVG masters and are not wired into the Studio UI.

## Creative direction

The identity uses a geometric Frame Bridge C: frame corners form the C, a central playhead represents frame-accurate editing, and cyan/violet sides suggest native engines meeting on one Chai Studio timeline. The emotional meaning of Chai is warmth, freshness, calm energy, and renewal without literal tea imagery.

## Prompt set

All samples used the built-in ImageGen path with `logo-brand` intent. Each prompt requested flat, vector-friendly geometry, a strong small-size silhouette, deep ink `#0A0F16`, cyan `#7ADCE8`, violet `#8D87FF`, accessible non-color-only state meaning, and no watermark. Prompts explicitly excluded teacups, tea leaves, steam, film reels, cameras, generic play triangles, clapperboards, copied third-party logos, and presentation mockups.

The individual prompts specialized the common direction for the primary mark, exact `Chai Studio` wordmark, macOS tile, exact `.chai` file extension, project thumbnail, six engine/bridge glyphs, nine operational states, six workspace empty states, six QA lifecycle marks, and three monochrome/accessibility treatments.

## Next gate
