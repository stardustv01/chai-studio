# Chai Studio Brand Concepts V3

Round 3 deliberately explores ideas beyond a forced letter C. These are ImageGen concept PNGs and local context tests, not production vectors, trademark clearance, or live UI integration.

## Concepts

1. **Chai Signature** - custom lowercase `chai`, technical `STUDIO`, and the `i` transformed into a playhead. Strongest emotional ownership and strongest wordmark direction. The isolated `i` is clean but needs more distinctiveness as an app symbol.
2. **First Light Frame** - fresh colour entering an incomplete frame. In context it reads too much like an edit or pencil icon.
3. **Warm Core** - living centre held by disciplined frame geometry. In context it drifts toward a camera or social-app icon.
4. **Two Currents** - cyan and violet paths remain distinct while meeting at one seam and opening into a shared frame. Strongest standalone symbol and closest to the product architecture.
5. **Living Cut** - one vertical edit cut separates unfinished and transformed frame states. Technically clear, but risks reading as a door, book, or split-screen control.
6. **Breathing Frame** - four open corners with intentional negative space. Calm and motion-friendly, but too close to crop/viewfinder UI language.

## Prompt set

All six concepts used the built-in ImageGen `logo-brand` path. Shared requirements were flat vector-friendly construction, recognition in dense professional software, the current deep-ink/cyan/violet palette, no watermark, and no literal tea imagery, film reels, cameras, play triangles, clapperboards, copied software logos, or presentation mockups.

The individual prompts isolated six different theses: an `i`-playhead wordmark, fresh light entering a frame, warmth held by discipline, two engine currents meeting, a transformative edit cut, and a frame that breathes through negative space.

## Context-test method

The generated concepts were placed in a local HTML comparison board and captured with the project-installed, isolated Playwright Chromium. Installed Chrome and user browser profiles were not used. The corrected board uses the actual isolated `i`-playhead crop for the Chai Signature Dock test.

## Current verdict

Round 3 produced a real direction rather than a single winning sketch: combine the **Chai Signature wordmark** with the **Two Currents architecture**. The next design should make the two currents meet at the same `i`-playhead seam used by the wordmark, creating one coherent identity across name, app mark, motion, and engine-bridge meaning. Do not integrate any current raster sample directly.
