# Chai Studio Icon Style Proof v1

Status: isolated artwork candidate — not integrated into the application.

This gate contains twelve representative, hand-authored SVG icons used to
approve the visual language before producing the full Chai Studio icon set.

## Included glyphs

- `workspace-edit`
- `workspace-media`
- `workspace-animation`
- `capture-exact`
- `render`
- `play`
- `blade-tool`
- `snap`
- `keyframe`
- `waveform`
- `qa-scan`
- `status-warning`

## Drawing contract

- 24 × 24 master grid
- 1.75 px stroke with round caps and joins
- `currentColor` for the primary drawing
- `--chai-icon-accent` for one restrained semantic accent
- reviewed at 14, 16, 20, and 24 px
- no embedded background or application-specific spacing

The specimen uses ivory on the midnight surface and ink on the warm-light
surface. Cyan marks precision, active playhead, and verification. Amber marks
authored warmth, media, and attention.

Open `specimen.html` through a local web server to inspect the live SVGs. The
rendered approval board is stored in `proof/style-proof-board.png`.
