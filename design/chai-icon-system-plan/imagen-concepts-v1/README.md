# Chai Studio Imagen Icon Direction v1

Status: visual-direction candidate — not production assets and not integrated.

`master-direction-board-v1.png` is a 24-icon Imagen concept board generated to
replace the rejected generic monoline direction. It establishes a richer
family language before producing the complete category sets.

`master-atlas-transparent-v1.png` is the corresponding background-free RGBA
test atlas. `transparent-samples-v1/` contains 24 individually isolated,
normalized 256 × 256 transparent PNG samples.

## What this direction establishes

- bold, distinctive silhouettes rather than generic outline-library forms;
- warm-ivory sculpted bodies on Chai midnight;
- cyan reserved for playheads, precision, active state, and verification;
- amber reserved for creative warmth and attention;
- controlled negative space and a subtle inset/enamel depth;
- recurring timeline, playhead, keyframe, and capture structures.

## Production constraint

Imagen artwork is a raster visual source, not automatically a production UI
icon. Approved forms must be generated consistently by category, isolated,
normalized, and converted into deterministic assets. Rich 20–24 px forms will
also need simplified 14–16 px optical variants.

The prior `full-set-v1` SVG candidate remains untouched for comparison but is
not the recommended visual direction.
