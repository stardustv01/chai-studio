# Example projects

`examples/manifest.json` indexes raw-media, Remotion, HyperFrames, mixed-engine, captions, audio, bridges/alpha, untrusted-import, and professional-edit examples. Each descriptor declares project class, assets, engines, and expected workflows.

Examples contain no private media and grant no executable trust. Tests materialize them into temporary `.chai` folders with deterministic fixtures and validate them through normal authority. They are documentation and regression inputs, never production projects or license grants.
